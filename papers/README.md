# Papers

The methodology is the contribution; the repository is its implementation. These
are the paper drafts the repository evolves alongside — each folder in the repo
corresponds to a section here. Draft in Markdown; export to LaTeX/PDF only when
submitting.

Planned sequence (each publishable on its own):

1. [`paper1-taxonomy.md`](paper1-taxonomy.md) — Why enterprise AI needs a
   deployment-readiness benchmark, and the banking trust taxonomy.
2. [`paper2-methodology.md`](paper2-methodology.md) — The synthetic-enterprise
   design and the layered, human-validated evaluation method.
3. [`paper3-results.md`](paper3-results.md) — First results and the
   judge-reliability study.

Discipline (from `docs/07`): research → peer review → revision → publication →
content. Nothing is amplified before it has survived scrutiny.
