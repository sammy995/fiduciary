# References

[`references.bib`](references.bib) is the canonical citation library — frameworks
(the source of ground truth for controls) and the evaluation / judge-reliability
methodology (the backbone). It is fed by the Phase-1 landscape review
([`../research/landscape.csv`](../research/landscape.csv)); add a BibTeX entry
whenever a paper enters the review, with its DOI/URL and a one-line research
note in the comment above it.
